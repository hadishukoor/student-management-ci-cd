# Student Management CI/CD Project

## Project Overview

This project demonstrates a complete CI/CD pipeline for a Java Spring Boot Student Management Application using:

- GitHub
- GitHub Actions
- Jenkins
- Maven
- Docker

## Features

- Add Student
- View Student List
- Responsive UI
- Automated Build and Deployment

## Technologies Used

- Java 21
- Spring Boot
- Maven
- Thymeleaf
- Docker
- Jenkins
- GitHub Actions

## Run Locally

```bash
mvn spring-boot:run
```

Application runs on:

http://localhost:8081
